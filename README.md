Hello! 

This is CRUD API project

 ## For first step - run
 ```
 npm i
 ```

 ## To build project run 
 ```
 npm build:prod
 ```

 ## To start in dev 
 ```
 npm run start:dev
 ```

  ## To start in prod 
 ```
 npm run start:prod
 ```

  ## To start with loader balancer

  ```
  npm run start:multi
  ```

  ## To testing:
  - 1 split console
  - 2 run  ```npm run start:dev``` or ```npm run start:prod``` or ```  npm run start:multi```
  in one console
  - 3 run ```npm run test``` in another console
